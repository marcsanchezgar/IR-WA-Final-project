Libraries required:

collections, math, numpy, pandas, json, datetime, time, array, nltk, re

To compile the the search engine use this command on this folder:

-python .\search-engine.py

Instructions will show in the console about how to do the diferent searches:

This are the instructions:
	Press 1 to search with tf-idf
	Press 2 to search with tf-itp (term popularity)
	Press 0 to exit (term popularity)
